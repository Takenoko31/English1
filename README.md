# English1

a16z / YC / Venture Deals / Think Fast Talk Smart 向けの、ディクテーション + リーディング学習アプリです。

## 機能

- コンテンツ切替（4カテゴリのサンプル文）
- リスニング + スクリプトを同一画面で表示
  - Spotify / YouTube の埋め込みプレイヤー表示
  - `<audio>` プレイヤー（音源URL設定時）
  - Web Speech API による代替読み上げ
- 公式サイト由来のスクリプト取得（取得可能なページのみ）
  - 「公式スクリプト取得」ボタンで読み込み
  - 取得失敗時は出典リンクへ移動して確認
- ディクテーション採点（簡易一致率）
- 語句選択からの辞書検索
  - 英英: dictionaryapi.dev
  - 英和: MyMemory Translation API
- 単語・表現の保存（LocalStorage）
- 保存表現のランダムテスト機能

## 使い方

1. `index.html` をブラウザで開く
2. 教材と文を選ぶ
3. 埋め込みプレイヤーまたは「音声読み上げ」で聴き取り、ディクテーション入力
4. 必要に応じて「公式スクリプト取得」で出典由来テキストを読み込む
5. スクリプトの語句を選択して「選択語句を調べる」または「選択語句を保存」
6. 保存後、「テスト開始」でランダム問題を実施

## 補足

- 現在はサンプル文を同梱しています。
- `script.js` の `contentData[]` で以下を設定できます。
  - `mediaEmbedUrl`: Spotify / YouTube の埋め込みURL
  - `mediaPageUrl`: 元ページURL（必要ならメモ用途）
  - `transcriptSourceUrl`: 公式スクリプトの参照元URL
  - `audio`: 直接再生する音源URL
- 外部サイトの仕様や CORS 制限により、スクリプトの自動取得は成功しない場合があります。
- 翻訳APIは無料エンドポイントを使うため、レート制限や不達が発生する場合があります。
