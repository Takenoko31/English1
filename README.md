# English1

a16z / YC / Venture Deals / Think Fast Talk Smart 向けの、ディクテーション + リーディング学習アプリです。

## 機能

- コンテンツ切替（4カテゴリのサンプル文）
- リスニング + スクリプトを同一画面で表示
  - `<audio>` プレイヤー（音源URL設定時）
  - Web Speech API による代替読み上げ
- ディクテーション採点（簡易一致率）
- 語句選択からの辞書検索
  - 英英: dictionaryapi.dev
  - 英和: MyMemory Translation API
- 単語・表現の保存（LocalStorage）
- 保存表現のランダムテスト機能

## 使い方

1. `index.html` をブラウザで開く
2. 教材と文を選ぶ
3. 「音声読み上げ」または音源再生で聴き取り、ディクテーション入力
4. スクリプトの語句を選択して「選択語句を調べる」または「選択語句を保存」
5. 保存後、「テスト開始」でランダム問題を実施

## 補足

- 現在はサンプル文を同梱しています。
- `script.js` の `contentData[].audio` に音源URLを入れると `<audio>` で再生できます。
- 翻訳APIは無料エンドポイントを使うため、レート制限や不達が発生する場合があります。
